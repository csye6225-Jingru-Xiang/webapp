# webapp

###Prerequisites
* Java 11
* Maven 3.8.6
* SpringBoot
* Postman
* Lombok Plugin
* MySql 8.0.28
* Navicat for MYSQL

###Build Instructions
1. Create database in mysql with 'localhost:3306'
2. Run the jar file in target folder
3. Hit the three APIs using Postman on port 8080:
   * /v1/user 
   * /v1/user/{accountId}
   * /v1/user/{accountId}
